Dahboard Preview
![Amazon_Prime_Dashboard](Amazon_prime_Dashbaord.PNG)

## Objective
To analyze Amazon Prime movies and TV shows data and build an interactive Power BI dashboard to understand content distribution, genres, ratings, and trends over time.

## Dataset
Amazon Prime titles dataset containing information about movies and TV shows such as genre, rating, release year, country, and type of content.

## Tools Used
- Power BI
- Excel / CSV Dataset

## KPIs
- Total Movies: 9655
- Total Directors: 5771
- Total Genres: 519
- First Movie Release Year: 1920
- Latest Movie Release Year: 2021

## Dashboard Insights
- Movies dominate the platform compared to TV shows
- Drama and Comedy are the most popular genres
- Content production increased significantly after 2015
- Majority of content falls under 13+ and 16+ ratings
