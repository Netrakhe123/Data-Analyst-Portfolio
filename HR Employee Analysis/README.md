Analyze employee data using SQL with simple joins and data cleaning.

- employees.csv
- departments.csv

- Identified and removed resigned employees

## SQL Concepts Used
- SELECT
- WHERE
- JOIN
- GROUP BY

## Key Analysis
- Mapped employees to departments
- Analyzed department-wise employee count
